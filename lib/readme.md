| 命令           | 说明                                                                                            |
| -------------- | ----------------------------------------------------------------------------------------------- |

| dts:package    | 生成 dts 类型定义文件                                                                           |
| prepublishOnly | 使用 [agadoo](https://github.com/Rich-Harris/agadoo)（另行安装）检查能否摇树优化 (tree-shaking) |
